## 📝 Description du Projet
AfriTalent est une plateforme web moderne et responsive conçue pour mettre en avant et connecter les compétences des freelances africains avec des entreprises locales et internationales. Le site propose une interface fluide offrant une expérience utilisateur premium, facilitant la recherche de profils qualifiés dans divers secteurs de l'économie numérique. Il intègre un système d'affichage dynamique, des animations interactives au scroll et une gestion adaptative de l'affichage.

---

## 🛠️ Technologies Utilisées
L'application repose sur un écosystème de technologies web standards, performantes et sans framework lourd :
* **HTML5** : Structuration sémantique avancée des pages.
* **CSS3** : Design System unifié basé sur des variables CSS personnalisées et architecture Flexbox/Grid (Bento Layout).
* **Bootstrap 5** : Framework CSS pour assurer une grille responsive et des composants d'interface natifs.
* **JavaScript (Vanilla JS)** : Logique applicative dynamique, manipulations du DOM, animations et algorithmes de filtrage.

## 🌟 Fonctionnalités Principales
1. **Mode Sombre & Clair Évolutif** : Basculement fluide de l'interface via un bouton dédié, adaptant l'ensemble de la charte graphique et mémorisant le choix de l'utilisateur (`localStorage`).
2. **Filtrage Dynamique des Freelances** : Système de tri instantané des cartes de profils par catégorie sans rechargement de page.
3. **Animations de Statistiques** : Compteurs digitaux animés se déclenchant de manière autonome lors de l'apparition à l'écran (`IntersectionObserver`).
4. **Effets d'Interface Dynamiques** : Navbar adaptative s'ajustant au défilement (effet *Shrink* & ombre portée) et bouton *Back to Top* intelligent pour une remontée fluide en haut de page.
5. **Validation de Formulaire Sécurisée** : Module complet de contrôle des données saisies dans le formulaire de contact avec gestion des états d'erreur en temps réel.
